Git

O Git é uma ferramenta usada para acompanhar todas as alterações feitas em um projeto. Ele registra cada modificação e permite voltar para versões anteriores, facilitando o trabalho em equipe e a organização do código.

GitHub

O GitHub é uma plataforma online onde podemos guardar, compartilhar e colaborar em projetos que usam Git. Ele funciona como um ponto central onde o código fica armazenado e acessível para outras pessoas da equipe.

Repositório

Um repositório é como uma pasta especial que guarda todos os arquivos do projeto junto com o histórico de alterações. É nele que tudo é organizado e mantido versionado.

.gitignore

O arquivo .gitignore serve para indicar ao Git quais arquivos e pastas devem ser ignorados. Isso evita que arquivos desnecessários, como temporários ou configurações pessoais, sejam enviados ao repositório.

README.md

O README.md é o arquivo principal de apresentação de um projeto. Nele ficam explicações sobre o objetivo, como funciona, como usar e informações importantes para quem for trabalhar ou consultar o projeto.
